# AI Agent Engineering Platform Frontend

Vue 3 + Vite 前端应用，用于展示 Agent 对话、RAG 知识库问答、思考链可视化和多模型配置。

## 功能

- Agent 流式对话
- 思考链和工具调用过程展示
- RAG 知识库文档管理
- DashScope / OpenAI / Anthropic / OpenAI-Compatible 模型配置
- Markdown 渲染、代码高亮和会话交互

## 启动

```bash
npm install
npm run dev
```

默认开发地址：

```text
http://localhost:3000
```

## 构建

```bash
npm run build
```

构建产物输出到 `dist/`，该目录不建议提交到仓库。
